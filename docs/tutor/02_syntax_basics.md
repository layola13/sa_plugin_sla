# 基础语法与类型系统

Sla 的语法风格非常贴近现代系统级语言（如 Rust 和 Zig），采用了表达式导向的设计，并具备严格、静态的强类型系统。

---

## 1. 变量与常量声明

### 变量绑定 (`let`)
Sla 使用 `let` 关键字进行变量绑定。对于一般的原始类型（如 `int`, `float`, `bool`），赋值与使用会默认复制值；而对于堆分配资源或不可复制的复杂结构（如 `Box<T>`, `Vec<T>`），赋值会直接**转移所有权** (Move)。
```sla
let a = 10;                     // 类型推导为 int
let b: i32 = 20;                // 显式指定 i32 类型
let my_box = Box::new(777);    // 拥有权类型
```

### 常量定义 (`const`)
常量在编译期确定其值，并且必须指定具体的类型。常量在整个程序生命周期内不可被修改。
```sla
const MAX_LIMIT: int = 100;
const PI: float = 3.14159;
```

## 2. 基本数据类型

Sla 拥有丰富的内置标量类型，它们与底层 SA 的硬件寄存器类型 1:1 映射：

| Sla 类型 | 底层 SA 映射 | 描述 |
| :--- | :--- | :--- |
| `i8` / `u8` | `i8` / `u8` | 8 位有符号/无符号整数 |
| `i16` / `u16` | `i16` / `u16` | 16 位有符号/无符号整数 |
| `i32` / `u32` | `i32` / `u32` | 32 位有符号/无符号整数 |
| `i64` / `u64` | `i64` / `u64` | 64 位有符号/无符号整数 |
| `isize` / `usize` | `isize` / `usize` | 架构相关平台指针位宽大小有符号/无符号数 |
| `int` | `i64` | `i64` 的简易别名 |
| `f32` / `f64` | `f32` / `f64` | 32 位/64 位浮点数 |
| `float` | `f64` | `f64` 的简易别名 |
| `bool` | `u8` | 布尔值类型 (`0` 代表 `false`，`1` 代表 `true`) |

## 3. 复合数据类型

### 元组 (Tuples)
元组是不同类型值的固定大小集合。元组的值通过括号包裹，并支持模式解构。
```sla
let pair: (int, bool) = (100, true);
let (val, flag) = pair;  // 解构变量绑定
```

### 固定大小数组 (Arrays)
固定大小数组在栈上连续存放同类型的值。
* **定义与初始化**：
  ```sla
  let arr: [int; 4] = [1, 2, 3, 4]; // 数组字面量
  let zeroed = [0; 10];             // 重复项字面量，长度为 10，初始值均为 0
  ```
* **读写访问**：使用中括号加索引访问（0 索引开始），支持运行时边界检查：
  ```sla
  arr[2] = 9;
  let element = arr[0];
  ```

### 结构体 (Structs)
结构体用于自定义复杂的数据布局。结构体定义中可以包含泛型参数，字段类型可以是原始类型或其它复合类型。
```sla
struct Point {
    x: int,
    y: int,
}

// 结构体实例化
let origin = Point { x: 0, y: 0 };
let current_x = origin.x; // 字段读取
```

### 枚举与有标签联合 (Enums & Tagged Unions)
枚举可以用来表示一组相关的变体，其中变体可以携带不同类型的数据成员（即代数数据类型 ADT）。
```sla
enum Message {
    Quit,                           // 无成员变体
    Move { x: int, y: int },       // 携带匿名结构体变体
}

// 实例化变体
let msg_quit = Message::Quit;
let msg_move = Message::Move { x: 10, y: 20 };
```\n
